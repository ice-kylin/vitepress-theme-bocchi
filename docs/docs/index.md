# 快速入门

简单、美观、易配置的 vitepress 主题。

## 1. 主题安装

1. 安装 `pnpm`：

   ```shell
   brew install node
   ```

2. 创建项目：

   ```shell
   mkdir vitepress-starter && cd vitepress-starter
   ```

   ```shell
   pnpm init
   ```

3. 安装依赖：
   ```shell
   pnpm i -D vitepress vite vitepress-theme-bocchi
   ```

4. 创建第一篇文档：

   ```shell
   mkdir docs && echo '# Hello VitePress' > docs/index.md
   ```
